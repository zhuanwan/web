# 可编辑文本

## 创建

使用 `new fabric.IText(string, object)` 方法创建可编辑文本

把文本添加到画布中，运行项目后，双击画布中的文本就可以进入编辑状态

点击画布空白位置或者点击键盘esc键可以退出编辑状态。


<Test1></Test1>

<br />

```js
const text = new fabric.IText('双击可编辑', {
  fill: '#f00',
  stroke: 'blue',
  strokeWidth: 2,
})
canvas.add(text)
```
